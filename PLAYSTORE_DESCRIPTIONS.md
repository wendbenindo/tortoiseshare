## DESCRIPTION COMPLÈTE (4000 caractères max)

**TortoiseShare - Partage Simple et Rapide**

Partagez facilement vos fichiers et votre écran entre votre téléphone Android et votre PC Windows, le tout via votre réseau WiFi local. Aucune connexion Internet requise, aucune inscription, aucune publicité.

**🚀 FONCTIONNALITÉS PRINCIPALES**

✓ **Transfert de Fichiers Rapide**
Envoyez vos photos, vidéos, documents et tout type de fichier de votre mobile vers votre PC en quelques secondes via WiFi.

✓ **Explorateur de Fichiers à Distance**
Parcourez tous les fichiers de votre téléphone directement depuis votre PC. Téléchargez ce dont vous avez besoin sans toucher à votre mobile.

✓ **Partage d'Écran Mobile**
Affichez l'écran de votre téléphone sur votre PC en temps réel. Idéal pour les présentations, le support technique ou simplement pour voir vos applications mobiles sur grand écran.

✓ **Envoi de Messages et Liens**
Partagez rapidement des liens et du texte entre vos appareils. Copiez facilement les messages reçus sur votre PC.

✓ **Connexion Automatique**
L'application mémorise vos connexions précédentes et se reconnecte automatiquement à votre PC pour un gain de temps.

**🔒 SÉCURITÉ ET CONFIDENTIALITÉ**

• Aucune donnée collectée - Zéro collecte de données personnelles
• Communication locale uniquement - Tout reste sur votre réseau WiFi
• Pas de serveurs externes - Connexion directe entre vos appareils
• Pas de publicité - Application 100% gratuite sans pub
• Open source friendly - Code transparent et sécurisé

**💡 COMMENT ÇA MARCHE ?**

1. Installez TortoiseShare sur votre mobile Android
2. Téléchargez l'application PC Windows (lien dans l'app)
3. Connectez les deux appareils au même réseau WiFi
4. Lancez les deux applications
5. Cliquez sur "Rechercher un PC" sur mobile
6. Sélectionnez votre PC dans la liste
7. C'est prêt ! Commencez à partager
